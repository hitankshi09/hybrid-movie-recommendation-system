# hybrid-movie-recommendation-system
Hybrid movie recommendation system that combines content-based filtering and collaborative filtering to generate personalized movie suggestions using the MovieLens dataset. The system provides real-time recommendations through an interactive Streamlit interface.

## Project Overview
This project implements a Hybrid Movie Recommendation System that suggests movies based on user input by combining content-based filtering and collaborative filtering techniques.
The system analyzes movie genres and user rating patterns to recommend movies similar to the selected title. It uses the MovieLens dataset to build a recommendation engine and provides an interactive user interface for generating recommendations.
This project demonstrates practical applications of Machine Learning and recommendation algorithms such as Collaborative Filtering.

## Features

- Hybrid recommendation model combining two filtering approaches
- Content-based filtering using movie genres
- Collaborative filtering using user rating patterns
- Real-time movie recommendations
- Interactive user interface built with Streamlit

## Dataset
The project uses the MovieLens dataset, which contains movie metadata and user ratings.

Dataset includes:
- Movie titles
- User ratings

This data is used to compute similarity between movies and identify user preference patterns.
